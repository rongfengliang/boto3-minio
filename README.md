# boto3 && minio python sdk  programing demo

## local running

* venv create

```code
python3 -m venv venv

source venv/bin/activate
```

* install dev package

```code
pip install -e .
```

* start minio server

```code
docker-compose up -d
```

* run with script

```code
boto3:

minioapp

minio:

minioc
```
